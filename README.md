# 🏦 Dashboard de Análise de Crédito - Estatística UFSJ

Este projeto foi desenvolvido como parte da disciplina de Estatística na **Universidade Federal de São João del-Rei (UFSJ)**.

## 📊 Sobre o Projeto
O objetivo é fornecer uma ferramenta interativa para análise exploratória de dados de crédito bancário, permitindo visualizar a dispersão de valores, o perfil de idade dos tomadores e medidas-resumo customizáveis.

## 🚀 Tecnologias Utilizadas
* **Python 3.x**
* **Streamlit** (Interface Web)
* **Pandas** (Manipulação de Dados)
* **Plotly** (Gráficos Interativos)

## 🛠️ Como executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o app: `streamlit run nome_do_seu_arquivo.py`
